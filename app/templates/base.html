Olá, seja bem vindo à nossa API Baby-Shower

O objetivo desta api é criar um marketplace em que pais e mães possam trocar ou vender produtos utilizados pelo bebê, como roupas, brinquedos, utensílios etc.

E como funciona? Esta api possui bancos relacionais que permitem armazernar conversas de chat entre os pais, produtos por eles postados, fazer perguntas relacionadas aos produtos de outros pais, bem como responder perguntas sobre seus produtos.

E possível filtrar produtos com uso de geolocalização, além de enviar emails automáticos ao enviar uma pergunta ou resposta.

Ainda existem filtros de nome do produto, preço, categoria e paginações.

Esta api possui poussui uma base de todas a cidades brasileiras.
